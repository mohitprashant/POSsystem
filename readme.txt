In order to run the program:

...The run.bat program as well as the classes folder must be placed in the same directory.

...The classes folder must contain the following .class files:
-Dessert.class
-Drink.class
-Item.class
-MainCourse.class
-Menu.class
-Order.class
-Promotion.class
-Reservation.class
-Restaurant.class
-RRPSS.class
-Staff.class
-Table.class
-Time.class

...Double click run.bat.
